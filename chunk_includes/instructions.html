Description de l'étude :
L'expérience inclut 32 situations. Pour chacune de ces situations, vous devez choisir entre deux phrases, celle qui selon vous correspond le mieux à ce qu'un locuteur du français quotidien dirait dans cette situation.

Lisez le contexte, puis les phrases et faîtes votre choix en cliquant sur les boutons pertinents. Cliquez SUIVANT pour continuer.

Avant de commencer l'expérience, vous aurez deux essais pour vous entraîner.

Quand vous aurez fini les questions, merci de remplir un bref questionnaire démographique. Cliquez SOUMETTRE VOS RÉPONSES pour finir l'expérience.

Si vous avez des questions
Vous pouvez contacter les chercheurs à deprez@linguistics.rutgers.edu.
